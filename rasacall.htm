01 — Project Overview

RasaCall is a conversational AI assistant built on the Rasa open-source framework, combining natural language understanding (NLU) with dialogue management for fluid multi-turn conversations. The system uses Rasa's pipeline architecture for intent classification and entity extraction.

The chatbot is configured with comprehensive training data including NLU examples for intent classification, domain definitions specifying entities, slots, and response templates, conversation stories for multi-turn flows, and rules for FAQ-style direct responses. The full pipeline is configured for production deployment readiness.

02 — Key Results
  • Complete Rasa NLU training data with intent classification examples
  • Domain definitions with entities, slots, and response templates
  • Conversation stories for multi-turn dialogue flows
  • Rule-based responses for FAQ-style interactions
  • Full pipeline configuration for production deployment
  • Both story-based and rule-based response modes for flexible conversation handling
