01 — Project Overview

This educational security tool demonstrates the practical differences in hash algorithm security by implementing both brute force and dictionary attack methods against MD5, SHA-256, and BCrypt hashes. It serves as a hands-on learning tool for understanding why modern hash algorithms like BCrypt are critical for password storage.

The brute force mode systematically tests character combinations with configurable character sets and length limits. The dictionary mode leverages a curated database of the top 10,000 most common passwords, demonstrating how weak passwords are vulnerable regardless of hash algorithm choice.

Built as a CLI tool with argparse for a clean command-line interface, making it suitable for security education workshops and demonstrations.
