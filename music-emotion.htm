Multimodal Video-to-Music Recommendation

An AI system that analyzes video content, classifies emotional context with a trained neural network, and recommends emotionally matching music from Spotify — bridging visual, emotional, and audio modalities.

01 — Project Overview

Matching music to video content is a fundamentally multimodal problem — it requires understanding visual scenes, mapping them to emotional categories, and finding audio that evokes the same feeling. This project builds an end-to-end pipeline that bridges all three modalities automatically.

The system uses Google Cloud Video Intelligence API to extract scene labels and context from video content, a trained Keras neural network to classify music tracks by emotional content based on Spotify audio features, and the Spotify Web API to retrieve playlist metadata. The emotion classifier model (emotion_classifier_model.h5) is trained on standardized audio features using scikit-learn's StandardScaler.

The result is a recommendation engine that takes any video as input and outputs a ranked list of emotionally appropriate music tracks from curated Spotify playlists.
