01 — Project Overview

PSAI-Bench is the first open-source benchmark designed specifically to evaluate AI systems on physical security alert triage — the task of looking at a camera-driven alert and deciding whether it's a real threat, suspicious activity worth a closer look, or a benign false alarm. Existing AI benchmarks measure language ability or vision capability in isolation; PSAI-Bench measures whether a system can actually reason across context the way a security operator does.

The v2.0 design uses context-dependent ground truth: the same description text resolves to THREAT, SUSPICIOUS, or BENIGN depending on zone type, sensitivity level, time of day, and device reliability. This kills the most common failure mode of v1.0 evaluation: single-field leakage, where a clever model could predict the verdict from description text alone with near-perfect accuracy. Real triage doesn't work that way.

The benchmark is system-agnostic. Bring your own system — LLM, vision model, rule engine, hybrid — produce verdicts in the required JSON schema, and PSAI-Bench scores them against the ground truth. Built-in evaluators for GPT-4o, Claude, and Gemini ship as reference integrations.

03 — Context-Dependent Ground Truth

The same alert description maps to different verdicts depending on the surrounding context. A system that classifies on description alone will be confidently wrong on the majority of scenarios — which is the entire point of the benchmark.

Description: "Person detected in restricted area" — verdict varies by zone × time

04 — Calibrated Metrics

Threat detection alone is a vanity metric — it rewards models that flag everything. PSAI-Bench scores each system on five dimensions so the trade-offs are visible.

TDR
Threat Detection Rate

Fraction of actual threats correctly classified as THREAT or SUSPICIOUS. Recall on the high-stakes class.

FASR
False Alarm Suppression

Fraction of benign alerts correctly classified as BENIGN. The dollar metric — this is what operators pay to stop seeing.

DEC
Decisiveness

Fraction of verdicts that commit to THREAT or BENIGN rather than hedging on SUSPICIOUS. Penalizes "I don't know" abuse.

ECE
Calibration Error

How well the system's confidence scores match its actual accuracy. Lower is better — required for downstream decisioning.
