01 — Project Overview

This tool automates the process of sourcing, downloading, and enhancing anime video clips from multiple platforms. It scrapes content from YouTube, Sakugabooru, Reddit (r/AnimeSakuga), and Tenor with intelligent rate limiting and category-based organization.

The standout feature is AI-powered super resolution using Real-ESRGAN, an anime-optimized generative model that upscales video frames at 2x, 3x, or 4x resolution with GPU acceleration. The system processes clips in batch with configurable JSON settings, organizing output by content category (action, fight, emotional, comedy).

Tech Stack Real-ESRGAN PyTorch yt-dlp BeautifulSoup OpenCV torchvision Python
02 — Key Results
  • Multi-source scraping: YouTube, Sakugabooru, Reddit r/AnimeSakuga, Tenor GIFs
  • Real-ESRGAN AI super resolution with anime-optimized model weights
  • GPU-accelerated upscaling at 2x, 3x, and 4x resolution options
  • Category-based organization: action, fight, emotional, comedy
  • Configurable via JSON with intelligent rate limiting per source
  • Batch processing utilities for handling large video collections efficiently
