01 — Project Overview

SEC filings contain rich business descriptions that are critical for sector classification, but manually categorizing thousands of filings across industries is impractical. This system uses a fine-tuned RoBERTa model with GPU-accelerated inference to automatically classify business descriptions into sectors, industries, and sub-industries.

The classifier implements a 10-method classification cascade with 18 rule categories, ensuring transparent fallback when the primary model's confidence drops below threshold. A SIC/NAICS code crosswalk maps 159 standard codes across 16 sectors, while semantic summarization achieves 60.8% text reduction without losing classification signal.

The system includes conglomerate detection via a 0.55 dominance ratio threshold, identifying companies that span multiple sectors rather than forcing single-sector assignment. Development followed a 5-phase lifecycle from data collection through production deployment.
